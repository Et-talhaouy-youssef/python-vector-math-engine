## 🤯 The "Why"
AI isn't magic; it's math. This engine is the result of a deep-dive into the mathematical foundations of LLMs and Neural Networks. It handles the core transformations that power modern AI systems, moving beyond simple scripts into professional, boxable software.
## 🔥 Key Features
- **✨ Vector Suite:** Operations for addition, scaling, and dot products.
- **🔳 Matrix Engine:** Handling transformations and dot products (Dimension-Checking logic included).
- **📦 Global Deployment:** Fully Dockerized for seamless execution on any machine.
- **🛡️ Error Handling:** Strict validation for mathematical dimension rules (No more out-of-bounds crashes).
